This project is a lightweight REST API built with Node.js to practice designing backend endpoints, handling query parameters, and structuring server-side logic.

The API provides travel location data based on query parameters supplied by the client, demonstrating server-side filtering and data retrieval.

---

## Project Goals
- Build a RESTful API using Node.js  
- Handle query parameters and return filtered data  
- Organize routes and utilities in a maintainable way  
- Practice clean server-side architecture and request handling  

---

## Features
- Query-based retrieval of travel location data  
- Server-side filtering logic based on user input  
- Modular route and utility structure  
- JSON-based API responses  

---

## Tech Stack
- Node.js
